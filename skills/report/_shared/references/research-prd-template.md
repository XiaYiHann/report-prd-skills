# 科研项目 PRD 模板（Research Program PRD）

在用户明确要"科研 PRD""研究计划书""grant proposal""项目申请书"或长期研究项目文档时，优先参考这份模板。

**核心理念**：战略蓝图 + 科学方法 + 风险管理。长度适中（10-30 页），活文档。

**最佳实践**：强调文献支持、伦理优先、可复制性。定期 review 更新假设。

---

## 设计原则

- **假设驱动**：每个核心主张都是可证伪的假设，而非不可质疑的断言
- **证据可见**：`claim -> evidence -> source -> limitation -> confidence`
- **风险前置**：伦理审查和 Go/No-Go 决策点在设计阶段就明确
- **影响闭环**：从研究问题到预期输出到传播计划的完整链条
- **迭代更新**：标记 Assumptions，定期 review

---

## 章节质量检查表

每章必须满足：

- [ ] 开头有明确结论句
- [ ] 支撑结论的要点 MECE
- [ ] 有至少一张图或表
- [ ] 假设是可证伪的，有明确证伪条件
- [ ] 证据层完整（claim -> evidence -> source -> limitation -> confidence）
- [ ] 图前有引导（说明图回答什么问题）
- [ ] 图后有结论（说明读者应带走什么判断）
- [ ] 无对照否定句（不是...而是...）
- [ ] 无纯描述性段落
- [ ] 无与其他章节重复的内容

**科研特殊检查：**

- [ ] 文献综述覆盖近 3-5 年关键工作
- [ ] Gap Analysis 明确指出机会
- [ ] Research Questions 是开放且可回答的
- [ ] Hypotheses 是可证伪的
- [ ] 实验设计包含基线、消融、失败案例分析
- [ ] 伦理考虑已覆盖（若涉及人类/敏感数据）
- [ ] 风险矩阵包含概率和影响评估
- [ ] Go/No-Go 决策点已定义
- [ ] Dissemination / Impact Plan 已包含

## 最简记忆口诀

**假设 → 证据 → 风险 → 影响**

或：

**可证伪的假设 → 可复制的实验 → 可评估的风险 → 可传播的影响**
