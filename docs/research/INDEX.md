# Research Framework Index

## Repository Role

This repository is the `research-loop` meta-framework repository. Its local
`docs/research/` directory is not a project-research workspace and must not
contain an active `CURRENT` pointer or `Vn/` epoch.

The local research documentation is limited to:

- framework charter and boundary notes;
- tracked agent policies used as templates;
- template notes that do not claim scientific results.

## Forbidden Local Content

- concrete project datasets, baselines, metrics, methods, or benchmark claims;
- repo-local paper binding decisions;
- generated PRD/Paper/Spec artifacts that look like project evidence;
- local `CURRENT` or `Vn/` state pretending this framework repository is a
  downstream research project.

## Agent Policies

- `agent/` contains behavioral policy templates for generated downstream
  research workspaces.
