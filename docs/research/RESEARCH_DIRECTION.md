## Research Seed

This repository is the research-loop framework repository. The local
`docs/research/` workspace is a generic scaffold for validating the research
workflow protocol, not a project-specific research direction.

No benchmark, method, dataset, baseline, or empirical claim is currently bound
to this workspace.

There is no repo-local active epoch. `CURRENT`, `Vn/`, PRD, Spec, Plan,
Task Queue, Baseline Lock, Paper Claim Ledger, and Paper Binding files belong
to downstream research repositories generated by this framework, not to this
meta-framework repository itself.

## Research Corridor

Allowed scope for this repository-level scaffold:

- Research-loop file protocol, schema, and state-machine validation.
- Skill, agent, installer, and documentation consistency.
- Evidence and anti-mock guardrail validation.
- Generic PRD, Spec, Plan, audit, insight, and paper-binding templates.
- Tests that verify the framework behavior without importing a concrete
  external research project.

## Out-of-Scope Directions

- Do not bind this repository scaffold to any external research project.
- Do not introduce project-specific datasets, baselines, methods, or claims.
- Do not treat generated placeholder documents as scientific evidence.
- Do not create paper results from unverified artifacts.
- Do not use this workspace as a benchmark result source.

## Prior Work Basis

No project-specific prior work is registered in this repository scaffold.
Downstream research projects must create their own `RESEARCH_DIRECTION.md`,
literature notes, reproduction records, and evidence ledger.

## Desired Paper Shape

No paper target is bound to this repository scaffold.

If a downstream project wants a paper target, it must be declared in that
project's own active epoch and approved by the owner.

## Autonomy Boundary

AI 可以自动做:

- generic research-loop scaffold consistency;
- validation scripts and tests;
- template wording that preserves claim boundaries;
- audit and evidence guardrails.

AI 不可以自动做:

- import project-specific research content into this repository scaffold;
- modify `RESEARCH_DIRECTION.md` to approve a concrete research direction;
- promote placeholder text to evidence;
- bind paper claims without observed artifacts and explicit human approval.
- create repo-local `CURRENT`, `Vn/`, or paper-binding artifacts for this
  framework repository.

## Global Stop Conditions

- The scaffold contains no project-specific research contamination.
- Framework format, schema, controller, installer, and validation tests pass.
- Downstream project-research content is kept outside this repository.
